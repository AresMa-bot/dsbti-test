MBTI、SBTI都已经过时了，DSBTI来了！一个比MBTI更有意思、更无厘头的人格测试！

## 特点

- 🎭 32种搞笑人格类型
- 📝 16道无厘头问题
- 🎨 清新简洁的界面
- 🤖 AI生成的低多边形风格图片
- 🎲 防重复结果机制，每次都不一样！

## 如何使用

### 方式1：直接打开
双击 `index.html` 文件直接在浏览器中打开即可。

### 方式2：本地服务器
```bash
# 使用Python启动本地服务器
python3 -m http.server 8080

# 然后在浏览器中打开 http://localhost:8080
```

## 部署到GitHub Pages

1. 在GitHub上创建一个新仓库
2. 将代码推送到GitHub
3. 在仓库的 Settings > Pages 中，选择 Source 为 `main` 分支，点击 Save
4. 等待几分钟，你的网站就会部署在 `https://你的用户名.github.io/仓库名/`
